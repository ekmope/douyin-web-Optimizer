# 抖音自动最高画质 (test_v7)

一个专注于画质提升的油猴脚本，自动将抖音网页版视频切换至当前可用的最高清晰度，具备**播放器就绪等待**、**快速滑动保护**、**画质只升不降**等机制，彻底解决手动切换的麻烦和脚本错乱问题。

## 功能特性

- ✅ **自动最高画质** – 检测到新视频后，自动点击清晰度选项中的最高档位，支持 4K/2K/1080p/720p 等。
- ✅ **播放器就绪保障** – 在操作前耐心等待播放器核心模块和码率选择器完全初始化，杜绝空操作。
- ✅ **快速滑动防丢** – 在异步等待的每一步都检查当前视频是否已经切换，一旦划走立即终止任务，避免为旧视频错误操作。
- ✅ **画质只升不降** – 记录全局“画质地板上限”，绝不把你手动调低的画质再拉回去；切换到 4K 后永久停止，不再干扰。
- ✅ **API + DOM 双重定位** – 优先通过播放器内部数据获取真实最高清晰度，若 DOM 未显示 4K 条目会自动刷新面板，直到找出隐藏的最高档位。
- ✅ **并发锁控制** – 快速滑动时不会同时运行多个升级任务，队列机制保证操作有序，杜绝面板闪烁。
- ✅ **宽松的选择器** – 使用模糊类名匹配面板和选项，降低抖音前段改版导致的脚本失效概率。
- ✅ **零依赖、纯净运行** – 脚本仅做画质切换，不包含任何隐藏元素、广告屏蔽等功能，避免与其它插件冲突。

## 安装方法

1. 确保浏览器已安装 [Tampermonkey](https://www.tampermonkey.net/) 或类似用户脚本管理器。
2. 点击脚本文件 `test_v7.js` 或复制其全部内容。
3. 在 Tampermonkey 中创建新脚本，粘贴代码并保存（`Ctrl+S`）。
4. 确保脚本已启用。
5. 访问 [抖音网页版](https://www.douyin.com/)，正常播放视频，脚本将在后台自动提升画质。

## 使用说明

脚本**全自动运行，无需任何配置**。

- 打开任意抖音视频页面或通过推荐流切换视频，脚本会自动将清晰度切换到当前允许的最高档。
- 如果你手动降低了某个视频的画质（例如从 1080p 切到 720p），后续视频的最高档依然会是 1080p 或更高，**不会再回到 720p**。
- 当你浏览到一个真正的 4K 视频且脚本成功切换到 4K 后，脚本会**永久停止**，直到你刷新页面或重新打开网站。
- 可在浏览器控制台看到前缀为 `[画质]` 的日志，输出当前状态和操作过程。

## 注意事项

- 本脚本仅适用于 **抖音网页版** (`www.douyin.com`)，不适用抖音直播、头条等其他平台。
- 脚本依赖当前播放器的 DOM 结构和 `window.player` 接口；若抖音大规模改版，可能需要更新 `findQualityBtn`、`findQualityItems` 等选择器或 API 调用逻辑。
- 极少数情况下（如网络波动导致播放器 API 未及时提供清晰度列表），脚本会放弃本次升级并输出日志，不会反复重试造成卡顿，下一个视频会自动恢复。
- 如果你使用了其他修改抖音界面的脚本或插件，请留意可能的冲突（例如面板类名被修改）。建议将本脚本置于较前执行顺序。
